# PIRLS 題組自動產生器使用說明

## 功能簡介
這個工具可以將 PaGamO 格式的 Excel 檔案自動轉換為 PIRLS 題組，並整合到現有系統中。

## 使用方式

### 1. 準備 Excel 檔案
確保 Excel 檔案符合 PaGamO 格式：
- 科目欄位包含「閱讀素養題組」
- 內容欄位有完整的文章段落
- 題目使用子編號（如 1_1, 1_2, ...）

### 2. 執行轉換
在專案根目錄執行：
```bash
node tools/excel-to-quiz.js "Excel檔案路徑" --id 題組編號
```

**範例：**
```bash
node tools/excel-to-quiz.js "PIRLS_PaGamO_題組 (66).xlsx" --id 45
```

### 3. 自動完成的工作
- ✅ 解析文章標題和內容
- ✅ 提取所有題目、選項、答案、提示
- ✅ 自動判斷題型（直接提取/直接推論/詮釋整合/評估批判）
- ✅ 更新 `data/questions.json`
- ✅ 更新 `index.html` 文章列表

### 4. 測試
轉換完成後，開啟瀏覽器測試：
```
http://127.0.0.1:8080/quiz.html?id=45
```

## 支援的格式
- 題目數量：不限（系統自動適應）
- 選項數量：最多 5 個（A-E）
- 段落數量：不限

## 注意事項
1. 題組 ID 必須是唯一的數字
2. 如果 ID 已存在，會覆蓋舊資料
3. 執行前請確保已安裝 xlsx 套件（`npm install xlsx`）

## 🚀 使用方式

### 教師端

**AI生成題組**:
1. 訪問: `ai-generate.html`
2. 密碼: `034711752`
3. 輸入主題、難度、字數、題數
4. 點擊生成，等待10-30秒
5. 預覽編輯，確認儲存

**結果**: 題組自動發布到首頁！

## 🔧 技術細節

**後端**:
```bash
# 安裝依賴
npm install dotenv

# 環境變數 (.env)
GEMINI_API_KEY=你的API_KEY

# 啟動伺服器
node server.js
```

**API路由**:
- `POST /api/generate-quiz` - AI生成
- `POST /api/save-quiz` - 儲存題組

**動態載入**:
- index.html會自動從questions.json讀取最新題組
